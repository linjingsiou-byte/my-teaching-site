# 簡化計算魔法學院 (Simplified Calculations Academy)

「簡化計算魔法學院」是特別為國小四年級學生設計的簡化計算（四則運算結合律與分配律應用）自學互動系統。專案使用 React + Tailwind CSS 打造，設計有加減魔法、乘除魔法、武功秘笈彙整與實戰演練四大板塊，引導孩子在填空點選中，快速建立湊整十、整百、整千的簡化數感。

---

## 🎮 學院學習板塊 (Tab 1 ~ 4)

1. **加減的魔法 🪄**：學習連加湊十、連減帶著符號大搬家、以及減去兩數之和的「生活購物扣款」魔法。
2. **乘除的魔法 ⚡**：背誦並活用乘法好朋友卡牌（`25 × 4`、`125 × 8`），並掌握乘除交換「先變小再變大」的撇步。
3. **武功秘笈整理 📚**：林老師的計算口訣總彙整，提供左右對比範例，方便學生複習與自我整理。
4. **實戰挑戰演練 🎯**：包含五大題混合式簡化計算魔法填空題，檢測自學成果。

---

## 🛠️ 技術與教學特色

* **即時彈出式迷思提示**：當學生選取錯誤的魔法數字時，網頁會即時以浮動對話框（Tooltip）提供針對該錯誤選項的「**認知迷思引導**」，說明為什麼這個數字不適合先計算。
* **多巴胺多色系 UI 標籤**：每個板塊都有專屬的代表色（粉色、綠色、紫色、橙色），透過醒目的色彩區分，提高自學的專注度與樂趣。
* **自適應平板配置**：針對平板電腦（如 iPad）設計友善的點選區域，並設有底部大按鈕導覽列，非常適合國小「生生用平板」的課堂教學使用。
