# 線段圖大冒險 (Segment Diagram Adventure)

「線段圖大冒險」是專為國小一至六年級學生設計的互動式數學學習遊戲。本專案將數學中的「線段圖」概念轉化為解題挑戰，引導學生在互動中學習分析題目數據，並配備了基於 Gemini API 的「AI 老師求解」與語音朗讀小功能。

---

## 🎮 冒險級別與核心題型 (Level 1 ~ 3)

1. **Level 1：初級冒險家 🧁 (適合1-2年級)**：合起來是多少（加法組合）、剩下多少（減法剩餘）、多多少/少多少（相差比較）。
2. **Level 2：邏輯挑戰者 🍪 (適合3-4年級)**：基準量與倍數的關係（幾倍是多少）、平分之後是多少（除法平分）、兩步驟問題（先算倍數再算總和）。
3. **Level 3：數學大師 🍰 (適合5-6年級)**：比較量是基準量的幾倍（分數倍數）、和倍問題（總和與倍數）、差倍問題（相差與倍數）。

---

## 🛠️ 技術與互動特色

* **純原生 SVG 動態繪圖**：所有線段圖全部由 HTML 內的 SVG 元件動態渲染，不依賴任何外部圖片載入，圖形邊緣完美銳利。
* **智慧型 AI 老師求解**：整合 Google Gemini API，能夠根據題目類型與學生的年級，以對應的親切口吻（例如：低年級使用可愛疊字與引導語）提供小提示，而不直接透露最終答案。
* **網頁語音合成 (TTS)**：支援一鍵朗讀 AI 老師的解題提示，方便低年級孩子或習慣聽覺學習的學生使用。
* **輕量級 Vanilla JS 實作**：捨棄 Babel 與 React runtime，保證在任何平板或離線環境下均能秒速加載。
