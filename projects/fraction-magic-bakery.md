# 分數魔法烘焙屋 (Fraction Magic Bakery Portal)

「分數魔法烘焙屋」是特別為國小二至四年級學生設計的互動式數學學習遊戲入口。整個專案將抽象的分數概念融入趣味的「魔法麵包烘焙」情境中，讓學生在製作糕點的遊戲任務中，自然而然建立完整的分數數感。

---

## 🎮 包含的子遊戲關卡 (1.0 ~ 5.0)

1. **1.0 單位分數 🧁 (適合二年級)**：認識分母為 2、4、8 等單位分數與幾分之一的實質意義。
2. **2.0 分數的認識 🍪 (適合三年級)**：學習簡單同分母分數的命名、大小比較與數值表徵。
3. **3.0 分數的加減 🍩 (適合三年級)**：同分母分數的簡單加法與減法合成/分解運算。
4. **4.0 進階分數 🍞 (適合四年級)**：學習帶分數與假分數的互換，以及分數的大小關係。
5. **5.0 等值分數 🍰 (適合四年級)**：透過切分視覺模型，體驗等值分數（如 1/2 = 2/4 = 4/8）的關係。

---

## 🛠️ 技術實作特色

* **動態環境感知跳轉**：入口網頁使用 `portal.js` 實現動態環境識別。在本地開啟時自動採用相對路徑離線運作，在雲端開啟時自動跳轉至對應的 GitHub Pages 線上儲存庫。
* **高畫質美術圖標**：採用烘焙屋主題圖示與互動按鈕，提供沉浸式學習體驗。
