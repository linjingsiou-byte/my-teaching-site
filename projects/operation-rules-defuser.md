# 運算規則拆彈專家 (Operation Rules Defuser)

「運算規則拆彈專家」是特別為國小四年級學生設計的四則運算規則互動式學習遊戲。本專案將四則運算的規則轉化為「拆除炸彈」的趣味任務，引導孩子在闖關中加深對運算順序的肌肉記憶，主動避開出題老師最愛設計的「甜美陷阱」。

---

## 🎮 包含的拆彈任務關卡 (Level 1 ~ 4)

1. **第一關：左左右右指令 🍊**：熟練加減或乘除同級運算時，必須「由左往右」排隊計算，拒絕直覺陷阱。
2. **第二關：老大先走指令 🍇**：掌握加減乘除混合時，乘除為第一優先（老大）計算的核心法則。
3. **第三關：盾牌拆解指令 🌲**：認識括號作為「無敵盾牌」的最高優先權，以及括號前是減號時拆括號的變號要領。
4. **第四關：真相只有一個 🌹**：挑戰乘除交換律與綜合判斷去括號變號規則的精細概念理解。

---

## 🛠️ 教學實作特色

* **引導式錯誤反饋**：點選錯誤選項時不只顯示扣分，還會提供詳細的「迷思解析」，用口語化的語句向孩子說明為什麼這個直覺算順序會踩到地雷。
* **拆彈心法複習**：完成挑戰後會展示「四則運算心法大統整」，方便課堂教師進行統整與收斂教學。
* **響應式多巴胺配色**：以 Tailwind CSS 打造大方、活潑的高對比多巴胺配色卡片，並支援行動裝置與平板電腦流暢點選。
