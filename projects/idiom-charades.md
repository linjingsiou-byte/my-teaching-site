# 成語猜猜樂 (Idiom Charades)

「成語猜猜樂」是一款專為國小中高年級學生設計的合作型語文學習遊戲。遊戲採用「我說你猜」的互動機制，適合雙人一組在教室現場使用 iPad 進行挑戰。說題者需要看著螢幕上的成語與解釋，透過言語或動作向猜題者傳遞線索，但絕對不能說出成語中的任何一個字。

---

## 🎮 六大成語主題篇章

1. 🐶 **動物植物篇**：包含守株待兔、畫蛇添足、狐假虎威等經典動植物成語。
2. 🔢 **數字成語篇**：包含一心一意、三心二意、七上八下等含數字的豐富詞彙。
3. 🌤️ **自然景象篇**：包含風和日麗、山明水秀、乘風破浪等形容天地景色的成語。
4. 📜 **歷史故事篇**：包含臥薪嘗膽、三顧茅廬、負荊請罪等歷史典故成語。
5. 📖 **寓言故事篇**：包含刻舟求劍、掩耳盜鈴、揠苗助長等寓意深遠的寓言故事。
6. 🌟 **綜合挑戰篇**：包含半途而廢、同甘共苦、不可思議等生活中常用的成語。

---

## 🛠️ 技術實作與設計亮點

* **成語大師 AI 助教**：集成 Google Gemini API。當說題者不知道該如何用動作或無字方式描述成語時，可點選「✨ 呼叫成語大師助教」，AI 會給予「不含答案字」的比手畫腳動作提示、生活大搜查情境、以及挖空的填空造句，引導猜題者說出詞彙。
* **網頁語音合成 (TTS)**：支援一鍵將 AI 助教提供的口頭線索朗讀播放，方便低中年級學生或聽覺學習型孩子使用。
* **高感官回饋與動畫**：配備倒數計時進度控制、答對時的 Canvas Confetti 繽紛紙花特效，以及大字重壓顆粒感的 dopamine style 按鈕。
* **完全離線相容**：基於 HTML5、Tailwind CSS 與純原生 Javascript，無任何模組載入或 React 建置依賴，完全支援 iPad 在無網路或區域網路下的 `file://` 協議流暢運行。
