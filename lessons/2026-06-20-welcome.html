歡迎來到我的個人教學網站！開啟你的網頁設計之旅

📅 發布日期: 2026-06-20 🏷️ 分類: 網頁設計

你好！這是我個人的教學紀錄空間。在這裡，我將會定期發布關於前端開發、UI/UX 介面設計、Canva 實用技巧以及各種網頁設計的實務心得。

不論你是剛接觸程式的超級新手，還是想精進設計質感的開發者，希望這裡的內容都能對你有所幫助。

🎯 為什麼我要建立這個網站？

在多年的教學與接案過程中，我發現許多學習者面臨以下兩個痛點：

  1. 「懂程式但不懂設計」：寫出來的網頁功能很強，但介面看起來很陽春，不夠吸引人。

  2. 「懂設計但不懂程式」：在 Canva 或 Figma 設計出精美的畫面，卻不知道如何把它轉化為真正的網頁。

這個網站的成立初衷，就是為了解決這個鴻溝。我會分享如何將美學設計與乾淨的程式碼結合，打造出像 Apple 官網那樣充滿質感的網頁。

🗺️ 接下來的學習地圖

在這個網站中，我將文章分為以下幾個主要方向：

  • 🎨 視覺設計與 Canva 技巧：如何利用 Canva 快速做出有質感的網頁排版，並學會色彩搭配與字型美學。
  • 💻 前端程式基礎：從最簡單的 HTML & CSS 開始，一步步學會響應式佈局與 JavaScript 互動。
  • 🛠️ 實戰專案演練：模仿世界級經典網頁（如 Apple、Stripe 等）的風格，親自動手重構網頁。
  • 🚀 部署與工具：如何使用 Git & GitHub Pages 免費發布你的網站，讓全世界都看得到。

✉️ 開始你的第一步

💡 學習網頁設計最好的方式就是「動手做」。
